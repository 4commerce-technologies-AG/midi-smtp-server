# Security Policy

In case of an overall API compatibility we just support the latest release
with security updates.

#### Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.x.y   | Yes                |
| 2.3.x   | Yes                |
| < 2.3   | No                 |

#### Reporting a Vulnerability

In case of a vulnerability please report a security advisory.
